# GitCredit.dev

> CLI wrapper around git for contribution graph outside of GitHub

```sh
alias g="gitcredit"
g status
# git status results
g add .
g commit -m "changes"
g push origin main
# gitcredit activity +1

OR

g record
# gitcredit activity +1
```

see your contributions and follow others at <a>gitcredit.dev</a>
